# Mini-Project-FAI-Sem2
Low-Code Chatbot Deployment 
**Project Description: Job Pre-Screening Chatbot using WotNot**

This project focuses on developing an automated job pre-screening chatbot using the WotNot platform. The main objective of the chatbot is to simplify and streamline the initial recruitment process by collecting essential candidate information in an interactive and efficient way.

The chatbot interacts with users by asking a series of structured questions such as the candidate’s name, the role they are applying for, their experience level, and key skills. It uses message nodes, button options, and input fields to guide the user smoothly through the conversation.

All the collected data is stored in a Google Sheets database, where important variables like name, phone number, and skills are recorded for further evaluation. This allows recruiters to easily access and review candidate details without manual effort.

The chatbot ends the interaction by displaying a confirmation message such as “We’ll contact you,” ensuring a complete and user-friendly experience.

Overall, this project demonstrates how chatbot automation can improve efficiency in recruitment by reducing manual work, saving time, and providing a seamless user experience.
